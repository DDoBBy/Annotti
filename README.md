# Annotti
Data annotation tool for deep learning

## Code structure
```
  ├─ main         : Related to main process
  ├─ renderers    : Related to renderer processes
  ├─ commons      : Used for both processes
  ├─ templates    : html files
  ├─ resources
  │   ├─ imgs
  │   └─ styles   : css files
  ├─ main.js      : main process
  └─ package.json : npm guide
```
